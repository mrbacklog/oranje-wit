## Legacy (niet meer bijwerken)

De onderstaande mappen bevatten de originele prototypes per traject. Ze worden niet meer bijgewerkt — alle ontwikkeling gaat via `ti-studio/`.

- `speler-weergave/` → gemigreerd naar `ti-studio/speler/`
- `staf-weergave/` → gemigreerd naar `ti-studio/staf/`
- `team-weergave/` → gemigreerd naar `ti-studio/team/`
- `navigatie/` → gemigreerd naar `ti-studio/shell/`
- `shared/` → gemigreerd naar `ti-studio/shared/`
