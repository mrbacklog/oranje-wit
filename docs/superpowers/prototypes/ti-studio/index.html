Shared Modules
Gedeelde tokens + JS-modules voor alle prototypes
tokens.css
Alle design tokens, avatar primitives, memo-attention, team-kaart CSS, staf-compact, scroll utility.
shell.js
Ribbon + grid-layout als JS-module. Elke pagina importeert shell() — wijzig de ribbon op één plek.
icons.js
Lucide SVG-paden: compass, briefcase-medical, shield, shirt, memo. Render helpers: svg(), memoSvg(), spelerBadge().
staf-components.js
stafChip(), stafRijkeRij(), stafSectie() — staf-elementen als herbruikbare functies.
