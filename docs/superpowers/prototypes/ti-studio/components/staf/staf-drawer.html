Staf-drawer

Werkbord zijpaneel · Rijke-rij kaarten met zoek/filter · Inactief filter toont gedempte kaarten

Zelfde structuur als de SpelersPoolDrawer: header met titel + count, zoekbalk, filterchips. De lijst bevat rijke-rij kaarten (vierkante foto + naam + rol-mini). Het "Inactief" filter toont inactieve stafleden met verlaagde opacity, grijze border en donkere foto.
