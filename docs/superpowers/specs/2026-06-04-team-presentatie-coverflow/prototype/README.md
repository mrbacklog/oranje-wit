# Prototype — Team-presentatielaag (Coverflow)

Visuele mockups uit de brainstorm-sessie. Open de `.html`-bestanden in een browser.

- **`coverflow-groot.html`** — ✅ goedgekeurde eindversie (grote schermen).
- `coverflow-v2.html` — compactere coverflow (3 leesbaar, geen overlap).
- `detailpaneel.html` — volledige inhoud van één center-kaart.
- `filmstrip-filter.html` — filterbalk-opzet.
- `twee-in-beeld.html` — band vrij houden → 2 teams half/half (vergelijken).
- `nav-aanpak.html` — de 3 oorspronkelijke navigatie-opties (keuze-context).

Het carrousel-effect zit in de zelfstandige `<style>`-blokken (`.ow ...`) en rendert los
correct; alleen de buitenste pagina-chrome is ongestyled buiten de visual companion.

Zie `../HANDOVER.md` en `../../2026-06-04-team-presentatie-coverflow-design.md`.
