## 1. NBA/NCAA Basketball — Talent Assessment

### 1.1 NBA Draft Combine — Fysieke Metingen

De NBA Draft Combine is het meest gestructureerde beoordelingsmoment voor 18-22 jarige basketbaltalenten. De Combine evalueert prospects op de volgende categorieën:

**Antropometrie (lichaamsbouw):**

| Meting | Wat het meet | Relevantie |
|--------|-------------|------------|
| Lengte (met/zonder schoenen) | Absolute lengte | Positie-geschiktheid |
| Gewicht | Lichaamsmassa | Fysieke volwassenheid |
| Wingspan (armspanwijdte) | Bereik relatief aan lengte | Verdediging, schotverstoring, rebounding |
| Standing reach (staande reikwijdte) | Maximaal verticaal bereik | Blokkeren, rebounding |
| Handlengte en handbreedte | Greep en balcontrole | Passing, schieten, balbehandeling |
| Vetpercentage | Lichaamscompositie | Conditie, fitheid |

> **Bron:** NBA.com Draft Combine Stats; LPS Athletic (2025); NBADraft.net Combine Winners & Losers (2025)

**Atletische tests:**

| Test | Wat het meet | Gemiddelden |
|------|-------------|-------------|
| Standing vertical jump (zonder aanloop) | Explosieve beenkracht vanuit stilstand | Guards: 30.8", Centers: 28.5" |
| Max vertical jump (met aanloop) | Explosieve kracht met snelheidsopbouw | Guards: 37.2", Centers: 32.6" |
| Three-quarter court sprint | Rechte-lijn snelheid (75 voet / ~23m) | ~3.1-3.3 seconden |
| Lane agility drill | Zijwaartse beweeglijkheid, richtingsverandering | ~11.1-11.7 seconden |
| Reactive shuttle run | Reactievermogen + laterale snelheid | ~2.9-3.2 seconden |
| Bench press (185 lbs / 84 kg) | Bovenkracht en spiervermogen | Guards: 10 reps, Centers: 13 reps |

> **Bron:** LPS Athletic NBA Draft Combine Stats (2025); NBA.com Combine database

### 1.2 NCAA/NBA Scouting Report — Spelerskwaliteiten

Naast de fysieke Combine worden prospects uitgebreid gescout op spelkwaliteiten. De standaard NBA scouting template bevat de volgende categorieën:

**Aanval (Offense):**

| Subcategorie | Beoordelingscriteria |
|-------------|---------------------|
| Schieten (Shooting) | Schotmechaniek, schotselectie, efficiëntie per zone, driepointers (benchmark: >38% = 2.5x meer kans op succes), free throw percentage |
| Balbehandeling (Ball handling) | Dribbelvaardigheid, bescherming, creëren van ruimte |
| Passen (Passing) | Visie, timing, creativiteit, assist-to-turnover ratio (>2.5 = significant hogere draftpositie) |
| Positiespel (Off-ball movement) | Vrijlopen, screens zetten/gebruiken, spacing |
| Scoren (Scoring) | Scoringsvariatie, score bij druk, clutch performance |
| Spelcreatie (Playmaking) | Vermogen om kansen te creëren voor zichzelf en teamgenoten |

**Verdediging (Defense):**

| Subcategorie | Beoordelingscriteria |
|-------------|---------------------|
| On-ball defense | Voetwerk, laterale snelheid, positie houden |
| Help defense | Roteren, communiceren, positie kiezen |
| Rebounding | Positie, timing, boxout, offensief reboundvermogen |
| Schotverstoring (Contesting) | Blokken, handen bij het schot, closing out |
| Steals/Intercepties | Anticipatie, passing lanes lezen, gamblers ratio |
| Verdedigingsveelzijdigheid | Meerdere posities kunnen bewaken (switchability) |

**Basketball IQ en Intangibles:**

| Subcategorie | Beoordelingscriteria |
|-------------|---------------------|
| Court vision (speloverzicht) | Lezen van het spel, anticiperen op bewegingen |
| Decision-making (besluitvorming) | Juiste keuzes onder druk, tempo bepalen |
| Game feel (spelgevoel) | Intuïtief aanvoelen van ritme, momentum, matchups |
| Coachability | Bereidheid om te leren, instructies opvolgen |
| Leiderschap | Teamgenoten aansturen, verantwoordelijkheid nemen |
| Wedstrijdmentaliteit | Reactie op tegenslag, body language, intensiteit |
| Consistentie | Prestaties over langere periode, niet alleen piekmoment |
| Toughness (mentale hardheid) | Fysiek contact opzoeken, reboundzin, verdedigingsbereidheid |

> **Bron:** Last Word on Sports — NBA Scouting Template (2024); CBS Sports Draft Scouting Reports (2024); Yahoo Sports — What NBA Scouts Look For (2024); Pro Skills Basketball — What Coaches Look For (2024); ESPN USA Basketball Scouting Reports (2025)

### 1.3 Sleutelbevinding: Assist-to-Turnover Ratio als IQ-proxy

Onderzoek uit de 2024 NBA draft class toont aan dat prospects met een assist-to-turnover ratio boven 2.5 gemiddeld 7.8 plekken hoger werden gedraft dan hun pre-draft ranking voorspelde. Dit bevestigt dat **besluitvorming en spelintelligentie** meetbaar zijn en zwaar meewegen in professionele scouting.

> **Bron:** Yahoo Sports (2024)

## 2. Athletic Skills Model (ASM)

### 2.1 Overzicht

Het Athletic Skills Model (ASM) is ontwikkeld door René Wormhoudt (S&C coach KNVB), Geert Savelsbergh, Jan Willem Teunissen en Keith Davids. Het motto is **"first the athlete, then the specialist"** — eerst een complete atleet vormen, dan pas sportspecifiek trainen.

Het ASM bestaat uit drie concentrische bouwstenen:

```
┌─────────────────────────────────────────┐
│         Conditions of Movement          │
│  ┌───────────────────────────────────┐  │
│  │      Coordinative Abilities       │  │
│  │  ┌─────────────────────────────┐  │  │
│  │  │  10 Basic Movement Skills   │  │  │
│  │  │      (Fundamental 10)       │  │  │
│  │  └─────────────────────────────┘  │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

> **Bron:** Wormhoudt et al. (2018), *The Athletic Skills Model: Optimizing Talent Development Through Movement Education*, Routledge; athleticskillsmodel.nl

### 2.2 De Fundamental 10 — Basisbeweegvormen

| # | Vaardigheid | Omschrijving | Korfbalrelevantie |
|---|------------|-------------|-------------------|
| 1 | **Balans (Balance)** | Evenwicht behouden bij steunen, glijden, draaien, rijden, zweven, hangen | Schotbalans, landingen, verdedigingspositie |
| 2 | **Duel/Grip** | Duelleren, trekken, duwen, tillen, dragen, grijpen van materialen of personen | Dekken, fysiek duel, rebound |
| 3 | **Voortbewegen (Locomotion)** | Alle vormen: lopen, rennen, kruipen, zwemmen, fietsen, schaatsen | Vrijlopen, spurten, verdedigend bewegen |
| 4 | **Springen (Jump)** | Hoogspringen, verspringen, dieptespringen, touwtje springen | Schotsprong, rebound, doorloopbal |
| 5 | **Roteren (Rotation)** | Rotaties langs lengte-, breedte- en diepte-as: salto's, draaien, raderen | Draaibeweging bij schot, pivots |
| 6 | **Hand-oog coördinatie** | Gooien, dribbelen, slaan met racket/stick, schijnbewegingen, fijne handmotoriek | Schieten, passen, vangen, feinting |
| 7 | **Voet-oog coördinatie** | Passen/schieten met voeten, dribbelen, trappen, schijnbewegingen | Minder direct, wel voetwerk bij verdedigen |
| 8 | **Klimmen (Climbing)** | Bewegen over/op objecten, wandklimmen, touwklimmen | Algemene kracht, proprioceptie |
| 9 | **Zwaaien (Swinging)** | Lichaam door de ruimte bewegen aan rekstok, paal, touwen | Kernstabiliteit, coördinatie |
| 10 | **Integratie** | Combinatie van alle bovenstaande in veelzijdige bewegingspatronen | De complete atleet |

> **Bron:** athleticskillsmodel.nl/en/fundamental10/ (2024)

### 2.3 Coördinatieve Vaardigheden (Coordinative Abilities)

| Vaardigheid | Omschrijving | Op 18 jaar |
|------------|-------------|------------|
| **Aanpassingsvermogen (Adaptability)** | Beweging aanpassen aan veranderende omstandigheden | Moet "af" zijn |
| **Evenwicht (Balance)** | Statisch en dynamisch evenwicht | Moet "af" zijn |
| **Koppeling (Coupling)** | Meerdere deelbewegingen vloeiend combineren | Moet "af" zijn |
| **Kinetisch differentiëren** | Juiste dosering van kracht en snelheid in beweging | Fijnafstelling nog mogelijk |
| **Ruimtelijke oriëntatie** | Positie bepalen in relatie tot veld, bal, mede- en tegenspelers | Moet "af" zijn, verfijning door ervaring |
| **Ritmisch vermogen** | Timing en cadans in beweging | Moet "af" zijn |

### 2.4 Condities van Beweging (Conditions of Movement)

| Conditie | Omschrijving | Op 18 jaar |
|---------|-------------|------------|
| **Beweeglijkheid (Agility)** | Snel van richting veranderen met controle | Op topniveau, verfijning mogelijk |
| **Stabiliteit** | Rompstabiliteit, kernkracht | Op topniveau, kracht nog groeiend |
| **Flexibiliteit** | Beweeglijkheid van gewrichten | Moet onderhouden worden |
| **Kracht en uithoudingsvermogen** | Fysieke capaciteit voor prestatie en herstel | Nog groeiend tot ~22-25 jaar |

### 2.5 Ontwikkelingsfasen ASM

| Fase | Leeftijd | Focus | Verhouding basis:sport |
|------|----------|-------|----------------------|
| **Basic** | 5-9 | Breed motorisch fundament, alle 10 basisvormen | 80:20 |
| **Transition** | 10-14 | Coördinatieve vaardigheden, "golden age" motorische ontwikkeling | 50:50 |
| **Performance** | 15-18 | Sportspecifieke toepassingen, wedstrijdcontext | 20:80 |

Op 18-jarige leeftijd:
- **Moet "af" zijn:** alle 10 basisbeweegvormen, alle coördinatieve vaardigheden
- **Nog in ontwikkeling:** absolute kracht, spiermassa, wedstrijdervaring, tactisch inzicht op topniveau
- **Verfijning:** kinetisch differentiëren (doseren), sportspecifieke toepassing onder druk

> **Bron:** Wormhoudt et al. (2018); athleticskillsmodel.nl; Eisenmann (2023) — The Athletic Skills Model: A Synopsis

## 3. LTAD — Long Term Athlete Development

### 3.1 Relevante Fase: Train to Compete / Train to Win (16-18+)

Het LTAD-model (Balyi & Hamilton) beschrijft zeven fasen van sportieve ontwikkeling. Voor 17-18 jarigen zijn twee fasen relevant:

**Train to Compete (16-18 jaar):**

| Competentie | Status op 18 jaar |
|------------|------------------|
| Sportspecifieke techniek | Geautomatiseerd, toepasbaar onder druk |
| Tactisch inzicht | Complexe systemen begrijpen en toepassen |
| Fysieke voorbereiding | Periodisering, sport-specifieke krachttraining |
| Mentale weerbaarheid | Omgaan met wedstrijddruk, selectie, tegenslag |
| Voeding en herstel | Zelfstandig managen |
| Wedstrijdritme | Optimale voorbereiding op belangrijke wedstrijden |
| Training-wedstrijd ratio | 50:50 (inclusief wedstrijdspecifieke training) |

**Train to Win (18+ jaar):**

| Competentie | Status op 18 jaar |
|------------|------------------|
| Individueel trainingsplan | Volledig gepersonaliseerd |
| Piekprestatie | Gericht op topprestaties in competitie |
| Fysieke en mentale volwassenheid | Bereikt of bijna bereikt |
| Training-wedstrijd ratio | 25:75 (competitie + wedstrijdspecifiek) |
| Ondersteuningsteam | Toegang tot fysiotherapie, psychologie, voeding |

> **Bron:** Balyi & Hamilton, LTAD Framework; Athletics Canada LTAD (2015); IYCA — Long-Term Athletic Development; Australian Institute of Fitness LTAD Guide (2023); Sport Science Insider LTAD Model (2024)

## 4. Handbal — Jeugdtalentherkenning

### 4.1 Multidisciplinair Identificatiemodel

Handbal is als balsport met veel lichaamscontact, schieten en positiespel een waardevolle referentie. De Duitse Handbalbond (DHB) hanteert jaarlijkse scoutingcamps voor U15 (meisjes) en U16 (jongens) met de volgende beoordelingsgebieden:

**Technisch:**

| Vaardigheid | Subcriteria |
|------------|-------------|
| Werpen/Schieten | Werpkracht, schotvariatie, schottechniek, precisie |
| Balbehandeling | Vangen, passen, dribbelen onder druk |
| Passeerbeweging | 1-tegen-1 acties, lichaamstäuschung, tempowisseling |

**Tactisch:**

| Vaardigheid | Subcriteria |
|------------|-------------|
| Aanvalsinzicht | Positiespel, ruimte creëren, timing van acties |
| Verdedigingsinzicht | Communicatie, schuiven, blokvorming, helpverdediging |
| Transitie | Omschakeling aanval-verdediging, fast break herkenning |

**Fysiek:**

| Vaardigheid | Subcriteria |
|------------|-------------|
| Snelheid | Sprintsnelheid, reactiesnelheid, actiesnelheid |
| Kracht | Werpkracht, kernkracht, duelkracht |
| Beweeglijkheid (Agility) | Richtingsverandering, laterale beweging |
| Uithoudingsvermogen | Wedstrijdbelasting volhouden |
| Sprongkracht | Verticale sprong, landing |
| Lichaamslengte en -bouw | Reikwijdte, fysieke volwassenheid |

**Psychologisch:**

| Vaardigheid | Subcriteria |
|------------|-------------|
| Wedstrijdmentaliteit | Presteren onder druk, competitiedrang |
| Veerkracht | Omgaan met tegenslag, terugkomen na fouten |
| Coachbaarheid | Openstaan voor feedback, leervermogen |
| Zelfvertrouwen | Durven beslissen, initiatief nemen |

> **Bron:** ResearchGate — A Multidisciplinary Identification Model for Youth Handball (2014); Frontiers in Sports — Selection Biases in Elite Youth Handball (2025); PubMed — Anthropometric and Performance Measures for Talent Detection (2009); Journal of Expertise — Talent Identification and Development Processes (2024)

## 6. Volleybal — Multidimensionaal Talentmodel

### 6.1 Multifactorieel Framework

Recent onderzoek (2024-2026) in volleybal heeft aangetoond dat talentidentificatie het beste werkt met een **multidimensionaal profiel** in plaats van losse tests. De drie kerndimensies:

| Dimensie | Componenten | Meetmethode |
|---------|-------------|-------------|
| **Musculoskeletal Fitness (MSF)** | Explosieve kracht, snelheid, beweeglijkheid, kracht, coördinatie | Verticale sprong, sprint, agility test, verspringen, grip strength |
| **Functional Motor Competence (FMC)** | Hand-oog coördinatie, perceptuomotorische coördinatie | Sport-specifieke vaardigheidstests |
| **Declarative Tactical Knowledge (DTK)** | Spelinzicht, besluitvorming, patronenherkenning | Video-gebaseerde tactische tests |

### 6.2 Essentiële Kenmerken volgens Elite Coaches (2026)

Een recente studie (Beldman et al., 2026) onder elite volleybalcoaches identificeerde:

| Categorie | Kenmerken |
|----------|-----------|
| **Fysiek** | Lengte, action height, atletisch vermogen (agility, coördinatie, power) |
| **Game sense** | Begrip van ruimte, tijd en momentum |
| **Technisch** | Passen, opzetten (setting), serveren, smashen, blokkeren |
| **Perceptief-cognitief** | Snelle besluitvorming, anticipatie, voorspelling van tegenstander |
| **Psychologisch** | Mentale hardheid, zelfvertrouwen, focus onder druk |

### 6.3 Sleutelbevinding: Tactisch Inzicht als Differentiator

De integratie van DTK (Declarative Tactical Knowledge) met fysieke en motorische maten gaf het beste voorspellende model voor het onderscheiden van elite vs. sub-elite jeugdspelers. Puur fysieke tests zijn **onvoldoende** voor talentidentificatie.

> **Bron:** MDPI Functional 9(1):29 — Multidimensional Approach to Talent Identification in Youth Volleyball (2024); Beldman et al. (2026), International Journal of Sports Science & Coaching — Essential Characteristics for Talent Identification in Volleyball; PMC — Assessing Technical Skills in Talented Youth Athletes: Systematic Review (2020)

## 7. Korfbal — KNKV Talentontwikkeling

### 7.1 KNKV Visie op Talentontwikkeling

De KNKV heeft in 2024 een vernieuwde visie gepubliceerd op teamsamenstelling en talentontwikkeling:

**Kernprincipes:**

| Principe | Uitwerking |
|---------|-----------|
| Plezier als basis | Plezier is de belangrijkste reden waarom kinderen (en volwassenen) naar een sportvereniging komen |
| Uitdaging = ontwikkeling | Kinderen zijn van nature nieuwsgierig; als ze uitgedaagd worden is de ontwikkeling optimaal |
| Veilig klimaat | Een veilig klimaat draagt bij aan gezonde fysieke en psychologische ontwikkeling |
| Niet te vroeg selecteren | Vóór 12 jaar is het vrijwel onmogelijk om te identificeren wie de getalenteerde spelers zijn |
| Brede ontwikkeling | Verenigingen worden aangemoedigd om spelers zo breed mogelijk op te leiden |
| Gelijke kansen | Elke speler verdient dezelfde faciliteiten, middelen, aandacht en trainingskwaliteit |

### 7.2 Selectiecriteria vanaf 12 jaar

Vanaf 12 jaar mag het prestatieaspect worden meegenomen in teamsamenstelling. De KNKV noemt:

| Criterium | Toelichting |
|----------|-------------|
| **Atletisch vermogen** | Coördinatie, snelheid, sprongkracht — bijzonder belangrijk |
| **Lengte + atletisch vermogen** | Voorwaarde voor topsport; kleine spelers moeten uitblinken in 1-op-1 |
| **Actiesnelheid** | Superieur aan pure sprintsnelheid op een atletiekbaan |
| **Timing en positiekeuze** | Superieur aan pure sprongkracht |
| **Aanvalskwaliteiten** | Minimaal één sterk aanvalsaspect (balbezit) |
| **Verdedigingskwaliteiten** | Minimaal één sterk verdedigingsaspect (niet-balbezit) |
| **Breed korfbalvaardighedenprofiel** | Brede ontwikkeling wordt boven specialisatie geplaatst |
| **Besluitvaardigheid** | "Durven beslissingen nemen" tijdens het spel |

### 7.3 KNKV Talentprogramma Structuur

| Programma | Leeftijd | Selectie |
|----------|----------|---------|
| Regionale Talent Centra (RTC) | U12-U15 | Regionaal scouting |
| Talentherkenning | U15 | Regionale scouting → nationale radar |
| Jong Oranje | U17-U19 | Nationale selectie |
| Oranje | Senioren | Internationale selectie |

> **Bron:** KNKV — Visie Teamsamenstelling en Fundament Talentontwikkeling (november 2024); KNKV — Teamsamenstelling & Talentherkenning — Bouwplan Korfbaltechnisch Beleid; KNKV — Selecteren vanaf 12 jaar (2024); KNKV — Talentontwikkeling Kennisbank; CSE Topsportacademie — Talentprogramma Korfbal

## 8. Geconsolideerd Framework: Alle Categorieën

Op basis van alle bovenstaande bronnen zijn dit de **zes hoofdcategorieën** met subcategorieën voor het beoordelen van een 18-jarig toptalent:

### 8.1 TECHNISCH (sportspecifiek)

Vaardigheden die direct met het spel te maken hebben.

| # | Subcategorie | Omschrijving | Bronnen |
|---|-------------|-------------|---------|
| T1 | **Schieten** | Schottechniek, precisie, variatie, afstandsschot | NBA, Handbal, Korfbal |
| T2 | **Passen** | Techniek, timing, korte/lange pass, creativiteit | NBA, Hockey, Handbal |
| T3 | **Vangen/Balbehandeling** | Aannemen, controle, behandeling onder druk | NBA, Handbal, Hockey |
| T4 | **Schotvariatie** | Meerdere schottypes beheersen (afstand, doorloop, penalty) | NBA, Korfbal |
| T5 | **Schijnbewegingen (Feinting)** | Misleiden van tegenstander, body fakes, shot fakes | NBA, Handbal |
| T6 | **Passeerbeweging (1-op-1)** | Voorbijgaan aan directe tegenstander | NBA, Handbal, Korfbal |
| T7 | **Verdedigingstechniek** | Dekken, channeling, juiste positie, blokken | NBA, Hockey, Handbal |

### 8.2 TACTISCH (spelinzicht)

Vermogen om het spel te lezen, te begrijpen en te beïnvloeden.

| # | Subcategorie | Omschrijving | Bronnen |
|---|-------------|-------------|---------|
| TA1 | **Speloverzicht (Court/Field vision)** | Het hele veld overzien, anticiperen op bewegingen | NBA, Hockey, Volleybal |
| TA2 | **Besluitvorming (Decision-making)** | Juiste keuze op het juiste moment, onder druk | NBA, KNKV, Volleybal |
| TA3 | **Positiespel** | Ruimte creëren, juiste positie kiezen, spacing | NBA, Handbal, Korfbal |
| TA4 | **Spelcreatie (Playmaking)** | Kansen creëren voor zichzelf en teamgenoten | NBA, Hockey |
| TA5 | **Transitie (Omschakeling)** | Snel schakelen tussen aanval en verdediging | Handbal, Hockey |
| TA6 | **Patronenherkenning** | Systemen lezen, aanvalsbewegingen herkennen | Volleybal, NBA |
| TA7 | **Spelintelligentie (Game IQ)** | Intuïtief aanvoelen van ritme, momentum, matchups | NBA, Volleybal |
| TA8 | **Communicatie op het veld** | Coachen, aansturen, richting geven aan teamgenoten | Hockey, Handbal |

### 8.3 FYSIEK (atletisch)

Lichamelijke capaciteiten die de basis vormen voor prestatie.

| # | Subcategorie | Omschrijving | Bronnen |
|---|-------------|-------------|---------|
| F1 | **Snelheid** | Sprintsnelheid, acceleratievermogen | NBA Combine, ASM, LTAD |
| F2 | **Actiesnelheid** | Snelheid van handelen in spelsituaties (boven pure sprint) | KNKV, Handbal |

## 9. Korfbal-specifieke Vertaling

### 9.1 De Zes Pijlers voor Korfbal Scouting

Op basis van alle bovenstaande research, vertaald naar de specifieke context van korfbal (gemengd, zonder dribble, paal+korf, vak-indeling):

#### Pijler 1: SCHIETEN (SCH)

| # | Subvaardigheid | Wat beoordelen | Referentie |
|---|---------------|----------------|-----------|
| SCH-1 | **Afstandsschot** | Techniek, precisie, reikwijdte, consistentie | NBA shooting mechanics |
| SCH-2 | **Doorloopbal** | Timing, afzet, richting, hand-oog coördinatie bij bewegend schot | NBA layup variety |
| SCH-3 | **Strafworp** | Techniek, routine, mentale stabiliteit onder druk | NBA free throw |
| SCH-4 | **Schotvariatie** | Aantal beheerste schottypes, links/rechts, afstandsvariatie | NBA scoring versatility |
| SCH-5 | **Schotselectie** | Alleen schieten bij goede kans, niet forceren | NBA shot selection |
| SCH-6 | **Scoren onder druk** | Presteren bij achterstand, beslissende momenten | NBA clutch performance |

#### Pijler 2: AANVAL (AAN)

| # | Subvaardigheid | Wat beoordelen | Referentie |
|---|---------------|----------------|-----------|

| AAN-2 | **Positiespel** | Ruimte creëren en benutten, diepte zoeken | NBA spacing, Handbal positiespel |
| AAN-3 | **Dreigen** | Schijnbewegingen, verdediger op verkeerd been zetten | NBA shot fakes, Handbal Täuschung |
| AAN-4 | **1-tegen-1** | Voorbijgaan aan directe tegenstander, passeerbeweging | NBA isolation, KNKV |
| AAN-5 | **Spelcreatie** | Kansen creëren voor anderen, het spel "maken" | NBA playmaking |
| AAN-6 | **Reboundaanval** | Positie onder korf, offensieve rebound, tweede kans | NBA offensive rebounding |
| AAN-7 | **Transitie (aanvallend)** | Snel omschakelen naar aanval, openspelen | Handbal fast break |

#### Pijler 3: PASSEN (PAS)

| # | Subvaardigheid | Wat beoordelen | Referentie |
|---|---------------|----------------|-----------|

| PAS-2 | **Overzicht** | Meerdere opties zien, veld "lezen" | NBA court vision |
| PAS-3 | **Besluitvorming** | Juiste pass op het juiste moment, tempo bepalen | NBA assist-turnover ratio |
| PAS-4 | **Creativiteit** | Onverwachte passes, no-look, korte-lange afwisseling | NBA creative passing |
| PAS-5 | **Vangen** | Aannemen in beweging, onder druk, slechte ballen verwerken | Handbal balbehandeling |
| PAS-6 | **Tellen (assist)** | Frequentie van beslissende passes | NBA assists |

#### Pijler 4: VERDEDIGING (VER)

| # | Subvaardigheid | Wat beoordelen | Referentie |
|---|---------------|----------------|-----------|
| VER-1 | **Dekken** | Directe tegenstander afstoppen, positie houden | NBA on-ball defense |
| VER-2 | **Onderscheppen** | Anticiperen op passes, passing lanes lezen | NBA steals |
| VER-3 | **Rebound (verdedigend)** | Positie onder korf, boxout, bal veiligstellen | NBA defensive rebounding |
| VER-4 | **Communicatie** | Tegenstanders doorgeven, schakelen, waarschuwen | Handbal, Hockey |
| VER-5 | **Blokken/Schotverstoring** | Schot verstoren zonder overtreding, timing | NBA shot contesting |
| VER-6 | **Helpverdediging** | Uithelpen bij doorbraak, schuiven, roteren | NBA help defense |
| VER-7 | **Transitie (verdedigend)** | Snel terugschakelen na balverlies | Handbal transition defense |
| VER-8 | **Verdedigingsveelzijdigheid** | Meerdere type tegenstanders kunnen dekken | NBA switchability |

### 9.3 Ontwikkelstatus op 18 jaar

Gebaseerd op ASM en LTAD, wat zou "af" moeten zijn versus "nog in ontwikkeling":

| Status | Subcategorieën |
|--------|---------------|
| **Moet "af" zijn (geautomatiseerd)** | SCH-1..4 (schottechniek), PAS-1 (pastechniek), PAS-5 (vangen), VER-1 (dekken), FYS-7 (coördinatie), FYS-8 (balans), MEN-1 (werkethiek) |
| **Op hoog niveau maar verfijnbaar** | AAN-1..4 (vrijlopen, positiespel, dreigen, 1v1), PAS-2..3 (overzicht, besluitvorming), VER-2..5 (onderscheppen, rebound, communicatie, blokken), FYS-1..6 (alle fysieke capaciteiten), MEN-2..6 (concentratie t/m wedstrijdmentaliteit) |
| **Nog in ontwikkeling (groeit met ervaring)** | SCH-5..6 (schotselectie, scoren onder druk), AAN-5..7 (spelcreatie, rebound, transitie), PAS-4..6 (creativiteit, assist), VER-6..8 (helpverdediging, transitie, veelzijdigheid), MEN-5 (spelintelligentie), MEN-7..10 (coachbaarheid, zelfvertrouwen, teamoriëntatie, creativiteit) |
| **Groeit tot ~22-25 jaar** | FYS-4 (absolute kracht), FYS-6 (sprongkracht), FYS-9 (lichaamslengte bij late groeiers), FYS-10 (herstelvermogen) |

### 9.4 Korfbal-unieke Aspecten

Korfbal verschilt van de referentiesporten op een aantal punten die de weging beïnvloeden:

| Korfbalspecifiek | Impact op beoordeling |
|-----------------|----------------------|
| **Gemengd** (mannen + vrouwen in elk team) | Communicatie en teamoriëntatie nog belangrijker |
| **Geen dribble** | Balbehandeling minder relevant, passen/overzicht veel belangrijker |
| **Vak-indeling** (aanval/verdediging gescheiden) | Beide rollen moeten beheerst worden, geen "specialist" |
| **Paal + korf** (i.p.v. backboard + basket) | Schotvariatie anders (doorloopbal essentieel, geen layup tegen bord) |
| **Persoonlijke verdediging** | Dekken (VER-1) is kerncompetentie, geen zone defense |
| **Wissel aanval/verdediging** | Transitie-intelligentie en conditie extra belangrijk |
| **Relatief kleine velden** | Actiesnelheid > pure snelheid, korte explosies |
| **Gering fysiek contact** | Duel/grip minder belangrijk dan in handbal/basketbal |
